## How to Add More Assets

### Recommended Free Sprite Sources:

1. **OpenGameArt.org**
   - URL: https://opengameart.org/
   - License: Various (check each asset)
   - Best for: RPG characters, enemies, backgrounds

2. **Kenney.nl**
   - URL: https://kenney.nl/assets
   - License: CC0 (Public Domain)
   - Best for: UI elements, simple characters

3. **Itch.io Free Assets**
   - URL: https://itch.io/game-assets/free
   - License: Various (check each pack)
   - Best for: Complete asset packs

4. **Game-Icons.net**
   - URL: https://game-icons.net/
   - License: CC-BY 3.0
   - Best for: Skill icons, UI icons

### Before Using Any Asset:

1. **Check the license** - Make sure it allows:
   - Commercial use (if you plan to monetize)
   - Modification (if you need to edit)
   - Distribution (for sharing your game)

2. **Give proper attribution** - If the license requires it (like CC-BY):
   - Add artist name to this file
   - Include link to original asset
   - State the license type

3. **Keep this file updated** - Add any new assets you download to this credits file
