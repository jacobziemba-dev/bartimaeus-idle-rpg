# AFK Arena Tech Stack

## Combat
- Auto-battle with manual ultimate activation
- 5-hero teams, 90-second battles
- Dual bars: HP + Ultimate meter
- Formation/positioning strategy matters

## Visual Style
- Stained glass + Art Nouveau aesthetic
- Painterly with bold lines, flat shading
- Ornate frames, glowing effects
- Character cards with faction colors/badges

## Game Loop
1. Summon heroes → 2. Level up → 3. Auto-battle → 4. Earn rewards → Repeat

## Idle Mechanics
- AFK chests accumulate resources (12hr intervals)
- Time-gated progression (not energy-based)
- Offline rewards on login

## Implementation Needs
- Animated character sprites (idle, attack, cast, death)
- Particle effects for skills
- Ornate UI frames and panels
- Ultimate meter system
- 90-second battle timer
- AFK chest collection UI
